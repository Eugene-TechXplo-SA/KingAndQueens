# マスター管理 / Master Settings

## 画面概要

| 項目 | 内容 | English |
|------|------|---------|
| 画面名 | マスター管理 | Master Settings |
| 目的 | サービス運営に必要なマスターデータ（ブローカー/配分比率/ランク/キャンペーン/参考レート/緊急停止等）を一元管理する | Centrally manage master data required for service operations |
| 主な利用ロール | ADMIN | |
| 責務 | ルール設定、ブローカー管理、配分比率管理、ランクマスター管理、キャンペーン管理、参考レート管理、緊急停止スイッチ操作 | |
| やらないこと | 個別ユーザーの付与計算（A-01）、承認タスク処理（A-02）、ユーザー個別情報編集（A-03） | |
| 画面構成 | ルール設定パネル + ブローカー一覧 + 配分比率パネル + ランクマスター + キャンペーン + 参考レート + 緊急停止スイッチ | |

## アクション定義

| No | トリガー | 処理内容 | 状態変更 | ログ有無 | 備考 |
|----|---------|---------|---------|---------|------|
| 1 | ルール設定「保存」 | デフォルト付与アセット・還元率を更新 | 設定値更新 | 必須 | |
| 2 | ブローカー「追加」 | 新規ブローカーをマスター登録 | Broker作成 | 必須 | |
| 3 | ブローカー「編集」 | ブローカー情報を更新 | Broker更新 | 必須 | |
| 4 | ブローカー「有効/無効切替」 | is_activeを切替 | Broker.is_active更新 | 必須 | |
| 5 | 配分比率「保存」 | アセット間の配分比率を更新 | DistributionRatio更新 | 必須 | 合計100%チェック |
| 6 | ランクマスター編集 | ランク名・倍率・昇格条件を更新 | RankMaster更新 | 必須 | |
| 7 | キャンペーン作成/編集 | キャンペーン情報を作成/更新 | Campaign作成/更新 | 必須 | |
| 8 | 参考レート更新 | 各アセットの参考レートを設定 | TokenRate更新 | 必須 | |
| 9 | 緊急停止スイッチ切替 | 取込/付与/出金/口座連携の停止/再開 | EmergencySwitch更新 | 必須 | 即時反映 |

## 状態遷移

| 状態 | 説明 | 遷移先 | 備考 |
|------|------|--------|------|
| ブローカー 稼働 | 有効。CSV取込対象 | メンテ中 | |
| ブローカー メンテ中 | 無効。CSV取込対象外 | 稼働 | |
| 緊急停止 ENABLED | 対象機能が有効 | DISABLED | |
| 緊急停止 DISABLED | 対象機能が停止中 | ENABLED | |
| キャンペーン 有効 | 期間内。倍率適用中 | 期間終了 | |
| キャンペーン 期間終了 | 期間外。倍率適用なし | - | |

## ガード条件

| No | ガード条件 | 内容 |
|----|-----------|------|
| 1 | Admin権限 | マスター管理操作はAdmin権限が必要 |
| 2 | 配分比率合計100% | 配分比率の合計が100%（=1.0）であること（BR-06） |
| 3 | ブローカーコード一意 | 同一コードの重複登録不可 |
| 4 | 参考レート正数 | レートは正の数値のみ |
| 5 | キャンペーン期間整合 | 開始日 < 終了日 |
| 6 | 還元率範囲 | 0〜100%の範囲内 |
| 7 | 倍率正数 | ランク倍率・キャンペーン倍率は正の数値のみ |
