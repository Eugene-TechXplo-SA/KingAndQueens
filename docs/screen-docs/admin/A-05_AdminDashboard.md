# 管理ダッシュボード / Admin Dashboard

## 画面概要


| 項目      | 内容                                       | English                                                           |
| ------- | ---------------------------------------- | ----------------------------------------------------------------- |
| 画面名     | 管理ダッシュボード                                | Admin Dashboard                                                   |
| 目的      | サービス全体のKPIとアクション必要件数を俯瞰し、運用状況を把握する       | Overview of service KPIs, pending actions, and operational status |
| 主な利用ロール | ADMIN                                    |                                                                   |
| 責務      | KPI表示（未承認件数、未処理出金、有効ユーザー数、KQレート）、推移グラフ表示 |                                                                   |
| やらないこと  | 個別の承認/却下操作（各画面で実施）                       |                                                                   |
| 画面構成    | KPIカード4枚 + 推移グラフ3枚                       |                                                                   |

| No  | 項目名             | 表示名            | English Label                | I/O | 備考            |
