# 管理者ログイン / Admin Login

## 画面概要

| 項目 | 内容 | English |
|------|------|---------|
| 画面名 | 管理者ログイン | Admin Login |
| 目的 | 管理者が認証情報を入力し、管理画面にアクセスするためのログイン処理を行う | Admin authentication to access the management console |
| 主な利用ロール | ADMIN | |
| 責務 | 管理者認証（メール/パスワード）、セッション/トークン発行 | |
| やらないこと | 管理者アカウントの新規作成（初期登録はシステム管理者が直接行う） | |
| 画面構成 | ブランドロゴ + ログインフォーム | |

## 表示項目定義（一覧）

| No | 項目名 | 表示名 | English Label | I/O | 備考 |
|----|--------|--------|---------------|-----|------|
| 1 | ブランドロゴ | KING and QUEEN 管理コンソール | KING and QUEEN Admin Console | O | |
| 2 | メールアドレス入力 | メールアドレス | Email Address | I | |
| 3 | パスワード入力 | パスワード | Password | I | |
| 4 | ログインボタン | ログイン | Log In | I | アイコン: Material Icons `login` |
| 5 | ログインエラーメッセージ | メールアドレスまたはパスワードが正しくありません | Login Error | O | 認証失敗時に表示 |
| 6 | 権限表示 | 権限: 管理者 | Role: Admin | O | ログイン後に表示 |
| 7 | ログアウトボタン | ログアウト | Log Out | I | ログイン後に表示 |

## 表示項目定義（詳細ドロワー）

該当なし

## アクション定義

| No | トリガー | 処理内容 | 状態変更 | ログ有無 | 備考 |
|----|---------|---------|---------|---------|------|
| 1 | 「ログイン」ボタン | 認証情報を検証し、Admin用JWT/セッションを発行 | 認証状態確立 | - | 以降のAPI呼び出しに認証トークンを付与 |
| 2 | 「ログアウト」ボタン | セッション/トークンを無効化 | 認証状態失効 | - | ログイン画面に戻る |

## 状態遷移

| 状態 | 説明 | 遷移先 | 備考 |
|------|------|--------|------|
| 未認証 | ログインフォーム表示 | 認証済み | |
| 認証済み | 管理画面操作可能 | 未認証（ログアウト） | |

## ガード条件

| No | ガード条件 | 内容 |
|----|-----------|------|
| 1 | AdminUser登録済み | 認証対象の管理者アカウントが存在すること |
| 2 | AdminUser有効 | AdminUser.status = ACTIVE であること（DISABLEDはログイン不可） |
