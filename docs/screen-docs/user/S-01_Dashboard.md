## 3. 項目定義


| No  | 項目名             | 表示名                                    | English Label           | I/O |
| --- | --------------- | -------------------------------------- | ----------------------- | --- |
| 1   | ウォレット接続状態       | Wallet Connected / Wallet Disconnected | Wallet Status           | O   |
| 2   | ウォレットアドレス       | 0x…（省略表示）                              | Wallet Address          | O   |
| 3   | アセット構成ドーナツチャート  | アセット構成                                 | Asset Composition Chart | O   |
| 4   | アセット別残高一覧       | 保有アセット                                 | Asset Balances          | O   |
| 5   | 現在のランク          | ランク                                    | Current Rank            | O   |
| 6   | ランクバッジ          | ランクバッジ（Member/Silver/Gold）             | Rank Badge              | O   |
| 7   | お知らせバッジ         | 未読件数バッジ                                | Notification Badge      | O   |
| 8   | ランクティアライン       | Member / Silver / Gold                  | Rank Tier Selector      | I   |
| 9   | ランクカード          | ランク詳細カード                                | Rank Detail Card        | O   |
| 10  | ランクカード名         | {ランク名}                                  | Rank Name               | O   |
| 11  | 次ランクまでの残りLEV_USD | Next: {ランク名} まであと ${lev_usd} | LEV_USD to Next Rank | O   |
| 12  | ランク進捗バー         | 進捗バー                                    | Rank Progress Bar       | O   |
| 13  | ランク特典カード（付与レート） | 付与レート | Reward Rate Multiplier | O   |
| 14  | ランク特典カード（出金手数料） | 出金手数料 | Withdrawal Fee | O   |

> **カード構成順序:** アセット構成 → 保有アセット → ランク
>
> **対応アセット（9種）:** KQ / BTC / ETH / USDT / XRP / USDC / JPYR / JPYC / IZAKAYA
