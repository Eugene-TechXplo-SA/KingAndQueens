## 3. 項目定義

> S-06 メニューから「ウォレット」で遷移する画面。
> Izakaya Wallet の接続状態と受取先アドレスを管理する。接続方式自体は TBD のため、画面では状態と導線を定義する。


| No  | 項目名         | 表示名                     | English Label              | I/O |
| --- | ----------- | ----------------------- | -------------------------- | --- |
| 1   | 接続状態        | 未接続 / 接続中 / 接続済み / 接続失敗 | Connection Status          | O   |
| 2   | 接続状態インジケーター | ステータスドット                | Status Indicator           | O   |
| 3   | ウォレットアドレス   | {address}               | Wallet Address             | O   |
| 4   | ネットワーク      | Polygon                 | Network                    | O   |
| 5   | ウォレット接続ボタン  | ウォレットを接続する              | Connect Wallet             | I   |
| 6   | ウォレット切断ボタン  | 切断                      | Disconnect Wallet          | I   |
| 7   | 接続失敗メッセージ   | 接続に失敗しました               | Connection Failure Message | O   |
| 8   | メニューに戻るボタン  | メニューに戻る                 | Back to Menu               | I   |


> No.3, No.4, No.6 は `CONNECTED` の場合のみ表示。
> No.7 は `FAILED` の場合のみ表示。
