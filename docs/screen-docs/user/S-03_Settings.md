## 3. 項目定義

> S-06 メニューから「設定」で遷移する設定専用画面。
> プロフィールは `S-10 プロフィール`、KYC詳細は `S-08 本人確認（KYC）`、ランク詳細は `S-13 ランク詳細` を参照。

### メールアドレス変更

|----|--------|--------|---------------|-----|
| 1 | 現在メールアドレス表示 | 現在のメール | Current Email | O |
| 2 | 新メールアドレス入力 | 新しいメールアドレス | New Email Address | I |
| 3 | メールアドレス変更ボタン | メールアドレスを変更 | Change Email Address | I |

## 4. アクション

| No | 対象項目No | 対象項目名 | トリガー | アクション概要 | 遷移先画面名 | 参照API | 備考 |
|----|-----------|-----------|---------|--------------|------------|--------|------|
| 1 | 3 | メールアドレス変更ボタン | タップ | 新メールアドレスの形式・重複を検証して更新する | - | | `email` 重複時は409 |
| 2 | 7 | パスワード変更ボタン | タップ | 現在パスワード確認後に新パスワードへ更新する | - | | パスワードポリシー検証あり |
| 3 | 10 | 退会申請ボタン | タップ | ガード条件を満たす場合のみ退会申請を送信する | - | | User.status を `WITHDRAW_REQUESTED` へ更新 |
| 4 | 11 | メニューに戻るボタン | タップ | メニュー画面へ戻る | S-06 メニュー | | |
| 5 | - | 画面読み込み時 | 画面表示 | 現在メールアドレスと退会可能状態を取得する | - | | 退会可否は残高・進行中出金申請で判定 |
