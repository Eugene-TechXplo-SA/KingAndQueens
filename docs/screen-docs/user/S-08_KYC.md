> **Note:** 書類種別選択・撮影・アップロード等の入力UIはすべてSumsub WebSDK内で提供されるため、独自の入力項目は不要。

### ステート別表示切替（switchKycState）

**未提出（NONE）:**

- No.1: fa-id-card（グレー）
- No.2: 「本人確認が必要です」
- No.3: 「出金申請するにはKYC認証を完了してください。」
- No.4: 全ステップ非アクティブ
- No.5: 表示
- No.6〜9: 非表示

**審査中（PENDING）:**

- No.1: fa-hourglass-half（イエロー）
- No.2: 「審査中です」
- No.3: 「書類を確認しています。しばらくお待ちください。」
- No.4: ステップ1完了、ステップ2アクティブ、ステップ3非アクティブ
- No.5〜6: 非表示
- No.7〜9: 非表示

**承認済み（APPROVED）:**

- No.1: fa-check-circle（グリーン）
- No.2: 「本人確認が完了しました」
- No.3: 「KYCが承認されました。全ての機能をご利用いただけます。」
- No.4: 全3ステップ完了
- No.5〜6: 非表示
- No.7: 表示
- No.8〜9: 非表示

**却下（REJECTED）:**

- No.1: fa-times-circle（レッド）
- No.2: 「本人確認が却下されました」
- No.3: 「書類を確認の上、再提出してください。」
- No.4: ステップ1完了、ステップ2失敗、ステップ3非アクティブ
- No.5〜6: 非表示
- No.7: 非表示
- No.8: 表示（Sumsubから返却される却下理由）
- No.9: 表示
