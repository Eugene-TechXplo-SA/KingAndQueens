## 3. 項目定義

> S-06 メニューから「付与履歴」で遷移する画面。
> 本画面は Ledger 起点の報酬履歴のみを扱い、出金履歴は `S-07 出金申請` 内の履歴セクションで扱う。


| No  | 項目名        | 表示名                           | English Label       | I/O |
| --- | ---------- | ----------------------------- | ------------------- | --- |
| 1   | 期間フィルター    | 期間                            | Date Range Filter   | I   |
| 2   | 種別フィルター    | 付与 / 取消 / 調整                  | Entry Type Filter   | I   |
| 3   | CSV出力ボタン   | CSV出力                         | Export CSV          | I   |
| 4   | 履歴カード一覧    | 付与履歴                          | Reward History List | O   |
| 5   | エントリ種別ピル   | 付与 / 取消 / 調整                  | Entry Type Pill     | O   |
| 6   | 発生日        | yyyy/mm/dd                    | Entry Date          | O   |
| 7   | 数量一覧       | +{数量} {アセット} / -{数量} {アセット}   | Asset Amounts       | O   |
| 8   | 参照ID       | 参照: {batch_id / reference_id} | Reference ID        | O   |
| 9   | ページング操作    | 前へ / 次へ                       | Pagination Controls | I   |
| 10  | 0件表示       | 付与履歴はまだありません                  | Empty State         | O   |
| 11  | メニューに戻るボタン | メニューに戻る                       | Back to Menu        | I   |


> 1つの履歴カードは、同一イベントに紐づく複数アセットをまとめて表示する。
> 金額の真実は台帳（Ledger）であり、取消・調整は追記型で表現する。

## 4. アクション


| No  | 対象項目No | 対象項目名      | トリガー | アクション概要             | 遷移先画面名    | 参照API | 備考                   |
| --- | ------ | ---------- | ---- | ------------------- | --------- | ----- | -------------------- |
| 1   | 1      | 期間フィルター    | 変更   | 指定期間の付与履歴を再取得する     | -         |       | `from` / `to` 指定     |
| 2   | 2      | 種別フィルター    | タップ  | 指定種別の履歴のみに絞り込む      | -         |       | 複数選択可否は下位設計          |
| 3   | 3      | CSV出力ボタン   | タップ  | 指定期間の付与履歴 CSV を出力する | -         |       | 範囲上限は TBD            |
| 4   | 9      | ページング操作    | タップ  | 次ページまたは前ページを表示する    | -         |       | `page` / `page_size` |
| 5   | 11     | メニューに戻るボタン | タップ  | メニュー画面へ戻る           | S-06 メニュー |       |                      |
| 6   | -      | 画面読み込み時    | 画面表示 | 直近の付与履歴一覧を取得する      | -         |       |                      |

| 表示タイプ | 対象項目No | 対象項目名    | 表示条件              | メッセージテキスト                | English Message                              | 備考     |
| ----- | ------ | -------- | ----------------- | ------------------------ | -------------------------------------------- | ------ |
| インフォ  | 10     | 0件表示     | 履歴が0件の場合          | 付与履歴はまだありません             | No reward history yet                        |        |
